HearthPacks

HearthPacks simulates a pack opening based on the Blizzard trading card game, Hearthstone.

HearthPacks is based on the actual statistics of pack openings and fetches card face images remotely when a pack is opened.

Technologies used: Unity3D, C#, JSON

Trade Genius

Trade Genius is a trade simulator for NBA teams!

Trade Genius runs your simulation through an NBA trade machine that checks the requirements for a trade to be successful under the 2011 CBA.

Technologies used: Java, Android Studio, SQLite

Autonomous Robot

Built to compete in a Robot Olympics competition amongst classmates and finished tied for first place.

Tested and programmed analog sonar sensors on a PIC using A2D. Proposed final robot was intended to use image processing via netbook, sonar, PI controller via encoders, and master and slave communications between PICs.

Technologies used: PIC, SPI, I2C, UART, C, MPLAB, Image Processing, A2D

Air Flow Controlled Inclination System

An inclination system in which a myDAQ controls the inclination angle of a plank.

Programmed and constructed a myDAQ in LabView to control an inclination angle of a plank by implementing a transfer function in a P, I, and PI controller.

Technologies used: LabView, Hall Effect Sensor, NI myDAQ
